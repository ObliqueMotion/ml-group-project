## How to Run

**Instal Dependencies**
* `pip3 install pygame`

**Run Game**
* `python3 game.py`

**Controls**
* `UP` 
* `DOWN` 
* `LEFT` 
* `RIGHT` 

**Exit Game**
* Press `Esc` on keyboard.
* Click the `x` on the window.
